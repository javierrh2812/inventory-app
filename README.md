# inventory-app

## requerimientos 
- node js 
- mongodb (instancia local)


## pasos para ejecutar

```sh
git clone https://github.com/javierrh2812/inventory-app
cd inventory-app
```

### cliente
```sh
cd vue_client
npm install 
npm run dev
```

### servidor
```sh
cd express-server
npm install
npm start
```
